# Forte Trader

Landing page da escola de day trade com Guilherme Andrade (Cabo Frio/RJ).

## Deploy no Vercel

1. Vá para [vercel.com/new](https://vercel.com/new)
2. Importe o repo GitHub: `pedrohpiri/Fortetrader`
3. Clique **Deploy** — pronto em segundos! 🚀

URL será algo como `fortetrader.vercel.app`.

## Desenvolvimento Local

```bash
# Abrir no browser
start forte-trader.html
```

## Tecnologias

- HTML5 + CSS3 + Vanilla JS
- Animações CSS + Canvas particles
- Totalmente responsivo
- SEO otimizado
